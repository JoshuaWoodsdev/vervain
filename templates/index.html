Welcome to Vervain

A Kadena Toolkit Built for Developers, by Developers

What is Vervain?

Vervain started as a simple idea: to make building apps on Kadena easier for everyone. It’s a homegrown toolkit designed to give you the basics you need to get started with Kadena today, while leaving plenty of room to grow into something bigger.

Right now, Vervain focuses on connecting your web apps to Kadena with simple tools to handle blockchain basics. As the toolkit evolves, it’ll bring even more features to make your projects smoother, faster, and more powerful.

What Can Vervain Do?

  • Simple Kadena Connections: Easily connect to Kadena’s blockchain and start interacting with it.
  • PACT Integration: Run basic interactions with PACT smart contracts right out of the box.
  • Wallet Support: Tools to connect wallets, letting your users sign transactions and interact with the blockchain directly.
  • Room to Grow: This is just the beginning! More features are on the way.

Why Build with Vervain?

  • Start Small, Dream Big: Vervain is built for developers who want to start with the basics and expand over time.
  • Made for Kadena: Kadena’s secure, scalable blockchain is the future of decentralized apps.
  • Stay Close to the Community: Built by developers, for developers. Your feedback shapes Vervain’s future.

What’s Next for Vervain?

Here’s what we’re working on for the future:

  • More powerful PACT tools for advanced contract management.
  • Pre-built UI components for blockchain-powered apps.
  • Debugging tools and performance insights tailored for Kadena.
  • Tutorials and examples to help you hit the ground running.

Let’s build something amazing, one block at a time.
